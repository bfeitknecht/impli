Browser Not Supported

Your browser doesn't support the following required features to run impli in the web.

  • JavaScript - required for WASM execution
  • WebAssembly - required to run the interpreter
  • ES6 Modules - required for loading dependencies

Please use a modern browser (Chrome 57+, Firefox 52+, Safari 11+, Edge 16+).

Alternatively, you can run impli from the command line. Check it out on GitHub!
