# Homebrew Formula

This directory contains the Homebrew formula for `impli`.

## Installation

To install `impli` using Homebrew from this repository:

```bash
brew install homebrew/impli.rb
```

Or add this tap to Homebrew:

```bash
brew tap bfeitknecht/tap
brew install impli
```

## Submitting to homebrew-core

To submit this formula to homebrew-core, follow the [Homebrew contribution guide](https://docs.brew.sh/How-To-Open-a-Homebrew-Pull-Request).

Note: The SHA256 checksum is automatically updated by the distribute.yaml workflow on tagged releases.
