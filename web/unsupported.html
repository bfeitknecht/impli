browser doesn't support all features required to run impli in the web.

Required Features

  • JavaScript
  • WebAssembly

  • Atomics
  • Service Workers
  • Cross-Origin Isolation

Minimum Browser Versions

  • Chrome/Edge: 92+ (July 2021)
  • Firefox: 95+ (December 2021)
  • Safari: 15.2+ (December 2021)
